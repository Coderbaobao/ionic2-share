# ionic2-share
一个ionic2动态分享应用
## 安装依赖
npm install 
## 启服务调试
ionic serve
## 添加android平台
ionic cordova platform add android
## 编译项目apk
ionic cordova build android

## 实现功能

- [x] 动态发布 （图文）
- [x] 留言点赞
- [x] 机器人聊天（暂时没有即使聊天功能）
- [x] 密码修改
- [x] 黑夜白天模式
- [x] 修改个人资料（头像、用户名、个新签名）
- [x] 发表记录
- [x] 查看关注的人
